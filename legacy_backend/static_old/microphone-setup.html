🎤 Fix Microphone Access

Follow these steps to allow microphone access for the voice agent

Step 1: Choose Your Browser

For Chrome / Microsoft Edge:

  1. Look at the address bar (where the URL is)
  2. Click the 🔒 lock icon or ⓘ info icon on the left side
  3. Find "Microphone" in the dropdown menu
  4. Change it from "Block" to "Allow"
  5. Click "Reload" or press F5
⚠️ Alternative Method:
1. Click the three dots (⋮) → Settings
2. Privacy and security → Site Settings
3. Microphone → Find localhost:1000
4. Change permission to Allow

For Firefox:

  1. Look at the address bar
  2. Click the 🛡️ shield icon or ⓘ info icon
  3. Click "Permissions"
  4. Find "Use the Microphone"
  5. Uncheck "Block" or select "Allow"
  6. Reload the page F5
⚠️ Clear Blocked Permissions:
1. Settings → Privacy & Security
2. Scroll to Permissions → Microphone → Settings
3. Remove localhost if blocked
4. Reload and allow when prompted

For Safari:

  1. Go to Safari menu → Settings (Preferences)
  2. Click the "Websites" tab
  3. Select "Microphone" from the left sidebar
  4. Find localhost in the list
  5. Change permission to "Allow"
  6. Close settings and reload the page
⚠️ macOS System Permissions:
1. Open System Preferences → Security & Privacy
2. Click "Microphone" tab
3. Make sure Safari is checked
4. Restart Safari if needed

Step 3: Return to TaxCalm

Once microphone access is granted:

  1. Go back to http://localhost:1000/
  2. Click the 🎤 microphone button in the chat header
  3. Start speaking - you should see your words appear!
✅ Pro Tips:
  • Use headphones to reduce echo and improve recognition
  • Speak clearly at a normal pace
  • Make sure you're in a quiet environment
  • If using Chrome, it works best with the voice agent
🔧 Still Not Working?
  1. Try using Incognito/Private mode (permissions reset)
  2. Check if other apps can access your microphone
  3. Update your browser to the latest version
  4. Try a different browser (Chrome recommended)
  5. Restart your computer and try again
